# Security Policy

## Supported Versions

Security fixes are applied to the latest release branch only.

## Reporting A Vulnerability

Do not post vulnerabilities, OAuth tokens, callback URLs, account IDs, or usage payloads in public issues.

Open a private GitHub security advisory for this repository, or contact the maintainer through the email address on the GitHub profile. Include the affected version, a minimal reproduction, impact, and any mitigation you have identified.

Reports are acknowledged after review. Please allow time for a fix before public disclosure.
