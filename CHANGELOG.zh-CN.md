## 0.1.40 - 2026-06-26

- 每次启动应用时都默认开启“显示 widget”，不再继承上一会话隐藏 widget 的状态。
- 移除桌面 widget 透明面板的原生阴影，修复外圈毛刺；保留 SwiftUI 的圆角柔和阴影。

## 0.1.39 - 2026-06-26

- 本版将“显示 widget”默认恢复为开启，即使升级前保留了旧版的关闭状态；在本版本中手动选择的开关状态仍会持续保存。
- 增加首个公开源码发布所需资料：可重复生成的 DMG 打包脚本、双语安装说明、安全策略和 Issue 表单。

## 0.1.38 - 2026-06-26

- 将“显示 widget”“置顶 widget”“重置位置”移回主窗口上方操作区域，避免下方设置说明被遮挡。
- 关闭“置顶 widget”时只降低 widget 层级，不再立刻把它重新置前。

## 0.1.37 - 2026-06-26

- 使用新的偏好键后，“显示 widget”默认开启。
- “置顶 widget”关闭时改用普通窗口层级，不再忽略鼠标事件，因此 widget 仍可拖动。
- 将“显示 widget”“置顶 widget”“重置位置”集中放在主窗口和菜单栏面板的同一组控制中。
- 通过“显示 widget”重新打开 widget 时，使用和“重置位置”一致的右下角默认位置。

## 0.1.36 - 2026-06-26

- 移除主窗口纵向滚动区域，让设置页更像固定的 macOS 工具窗口。
- 放大主窗口说明文字和额度元信息字号，提升阅读舒适度。
- 调整主窗口初始尺寸，使其和固定设置布局一致。
- 拖动 widget 不再打开主窗口；只有明确单击 widget 才打开主窗口。

## 0.1.35 - 2026-06-26

- “显示 widget”改为真正的持久化开关：关闭开关会关闭 widget，关闭 widget 也会同步关闭开关。
- “置顶 widget”关闭时，widget 保留在桌面层但不拦截桌面图标点击。
- 明确只有开启“置顶 widget”时才能直接拖动 widget。

## 0.1.34 - 2026-06-26

- 将 widget 操作文案从“固定 widget”改为“显示 widget”，让文案和实际行为一致。
- “显示 widget”改为幂等操作：重复点击只会置前并校正已有 widget，不再切换隐藏或重新创建。
- 增加“置顶 widget”设置：开启后 widget 浮在应用窗口前方，关闭后保留在桌面层。
- 保留“重置位置”作为明确的恢复操作：打开 widget 并放回屏幕可见区域右下侧。

## 0.1.33 - 2026-06-26
